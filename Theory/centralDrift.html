- Central drift for compressible fluid
\(\rho\) - Density
\(V\) - Specific volume
\(T\) - Temperature
\(p\) - Pressure
\(U\) - Ether speed
\(R\) - Radial coordinate
\(f\) - degrees of freedom
\(C_p\) - Specific heat at constant pressure
\(C_V\) - Specific heat at constant volume
We wish to calculate the steady state of a compressible ideal gas for is a single flow sink. The incoming flux (quantity per time) proportional to the mass of the object we mark the flux as \(KM\).
We will treat this gas as mon-atomic. Mon-atomic means that the only degrees of freedom of the particles of the gas are its linear motion. The is no rotational motions, and also no internal vibration. The number of degrees of freedom \(f\) is 3. The main effect of this is that every bit of energy we put into the gas, contributes directly to the pressure that this gas exerts on its containing walls. This gives a simple connection between the pressure and the density. We get \[\gamma=1+\frac{2}{f}=\frac{C_p}{C_v}=\frac{5}{3}\] For our case the flow is isentropic (no wall friction, no heat flow), so we get \[\left(\frac{T_2}{T_1}\right)^{\frac{\gamma}{\gamma-1}}=\frac{p_2}{p_1}=\left(\frac{\rho_2}{\rho_1}\right)^\gamma=\left(\frac{V_2}{V_1}\right)^\gamma\] Resulting in the state equation \[p\propto\rho^{\frac{5}{3}}\] What is this state equation?

How to solve the flow equations?
We will solve the Euler equation \[\rho U\frac{\partial U}{\partial R}+\frac{\partial p}{\partial R}=0\] First we use the state Equation \(p\propto\rho^{\frac{5}{3}}\) Which means that for some constant \(Q\) we have the connection \(p=Q\rho^{\frac{5}{3}}\).
We can then set the \(p\) in the Euler equation to get an equation with only the density \(\rho\) and the speed \(U\) \[\rho U\frac{\partial U}{\partial R}+\frac{\partial Q\rho^{\frac{5}{3}}}{\partial R}=0\] \[U\frac{\partial U}{\partial R}+Q\rho^{-1}\frac{\partial\rho^{\frac{5}{3}}}{\partial R}=0\]
Then use the identity
\[F(x)^\alpha\frac{\partial F(x)^\beta}{\partial x}=\frac{\beta}{\alpha+\beta}\frac{\partial F(x)^{\alpha+\beta}}{\partial x}\]
to get \[\frac{1}{2}\frac{\partial U^2}{\partial R}+Q\frac{5}{2}\frac{\partial\rho^{\frac{2}{3}}}{\partial R}=0\] \[\frac{\partial \left(U^2 + 5Q\rho^{\frac{2}{3}}\right)}{\partial R}=0\] Which means that \(U^2+Q^`\rho^{\frac{2}{3}}\) is a constant, so now in order to propose a solution we need to talk about boundary conditions.
The conditions we choose are that at a distance \(R_f\) (that is at \(R\) far away) we set the initial density \(\rho\) to \(\rho_f\). More than that, we make sure that the value of the density at \(R_f\) remains the same vy adding and removing particles as needed. So \(U^2+Q^`\rho^{\frac{2}{3}}=U_f^2+Q^`\rho_f^{\frac{2}{3}}\), and we can find \(U_f\) using \(\rho UR^2=KM\). So we get \[U^2+Q^`\rho^{\frac{2}{3}}=\frac{KM}{R_f^2\rho_f}+Q^`\rho_f^{\frac{2}{3}}\]
From the last two equations we get \[\frac{KM}{R^2}\frac{\partial U}{\partial R}+\frac{\partial p}{\partial R}=0\]
We can now make a guess that \(U(R)\propto R^\alpha\) for some \(\alpha\). From the last equation we get \[p(R)\propto R^{\alpha-2}\] And then from the state equation \[\rho(R)\propto R^{\frac{3}{5}\left(\alpha-2\right)}\] Collecting all the powers in to the continuity equation gives \[\frac{3}{5}\left(\alpha-2\right)+\alpha+2=0\quad\Rightarrow\quad\alpha=-\frac{1}{2}\] So the full solutions is
\[U(R)\propto R^{-\frac{1}{2}}\quad\quad a(R)\propto R^{-2}\] \[\rho(R)\propto R^{-\frac{3}{2}}\quad\quad p(R)\propto R^{-\frac{5}{2}}\] \[T(R)\propto R^{-1}\]
From this solution we see that \(\frac{U}{\rho R}\) is constant (marked \(\beta\)), and together with the continuity equation we get \[\rho UR^2=KM\quad\Rightarrow\quad\frac{U}{\rho R}\rho UR^2=\beta KM\quad\Rightarrow\quad U^2R=\beta KM\] Setting the constant \(\beta K\) to have a value of \(2G_N\) we get \[U^2R=2G_NM\] And this immediately give the final equation that makes gravitation
\[\frac{U^2}{2}-\frac{G_NM}{R}=0\]
