Auto-narrating this slide

Voice your
presentation.

Add data-narrate="..." to any slide element.
The Web Speech API reads it aloud on entry.

<!-- On the slide element: --> <section class="slide" data-narrate="Welcome everyone!"> <!-- Or on any element: --> <p data-tts="This is key insight #1"> Key insight #1 </p>

Press T to replay narration

Quick Reference

All data-attributes at a glance.

Animation Attributes

<!-- Position: translate X,Y --> data-pos-1="[100px, 50px]" data-pos-2="[300px, 50px]" <!-- Show/hide (opacity + class) --> data-show-1 data-hide-2 <!-- Opacity 0–1 --> data-opacity-1="0" data-opacity-2="1" <!-- Scale --> data-scale-1="0.5" data-scale-2="1.2" <!-- Rotation (degrees) --> data-rotate-2="45" <!-- Duration per element --> data-duration="800ms" <!-- Add CSS class at state N --> data-class-2="highlighted"

Charts & Narration

<!-- Chart types: bar line area pie donut --> data-chart="bar" data-chart-data='{"labels":[...],"values":[...]}' data-chart-color="#4fffb0" <!-- Reveal chart at state N --> data-chart-state="2" <!-- Slide narration (auto on entry) --> data-narrate="Hello everyone..." <!-- Element TTS (plays on T key) --> data-tts="This is important!" <!-- Navigation controls --> ↑ ↓ Space → Change slide ← → → Change state F → Fullscreen T → Narrate

Getting started

Building your first slide.

1

Include the files

Add deck.css and deck.js to your HTML page.

2

Wrap slides in .deck

Each <section class="slide"> is one slide.

3

Add data-attributes

Use data-pos-N, data-show-N, etc. on elements.

4

Present!

Open in any browser. No build step, no server needed.

<!DOCTYPE html> <html> <head> <link rel="stylesheet" href="css/deck.css"> </head> <body> <div class="deck"> <!-- Slide 1 --> <section class="slide" data-narrate="Hello world!"> <h1 class="slide-title">My Slide</h1> <p data-pos-1="[0px, 100px]" data-pos-2="[0px, 0px]" data-opacity-1="0" data-opacity-2="1"> Animates in on state 2 </p> </section> </div> <script src="js/deck.js"></script> </body></html>
